# Star Gazetesi Web Sitesi

Bu proje, Star Gazetesi'nin modern ve duyarlı web arayüzünü içermektedir.

## Özellikler

- Tam duyarlı tasarım (mobil, tablet ve masaüstü uyumlu)
- Modern ve şık kullanıcı arayüzü
- Haber kategorileri için özel bölümler
- Canlı yayın bölümü
- Sosyal medya entegrasyonları
- Hızlı yükleme süreleri

## Teknolojiler

- HTML5
- CSS3 / SCSS
- JavaScript (ES6+)
- Bootstrap 5
- Owl Carousel 2
- Font Awesome 6
